### Installation

1. Clone the project.
2. Run `npm install`
3. Set the client key env variable at `.env.local` like below:

```
VITE_GOOGLE_RECAPTCHA_CLIENT_KEY=6LDNE_spAAAAA....
```
4. Run `npm run dev` 
5. Open your browser at localhost:4010. Click the Generate button to generate a recaptcha key.
